「ONE K」のランキング報酬は以下となっています。  
順位によってコインを取得できます。  

  
<br>
  
|  順位  |  取得コイン  |
| ---- | ---- |
|  1位  |  40コイン  |
|  2位  |  20コイン  |
|  3位  |  10コイン  |
|  4~10位  |  5コイン  |

※今後のアップデートで取得数は変更になる可能性があります。
